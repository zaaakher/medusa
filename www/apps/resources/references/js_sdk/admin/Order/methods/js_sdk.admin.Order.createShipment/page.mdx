# createShipment

This method creates a shipment for an order's fulfillment. It sends a request to the
[Create Shipment](https://docs.medusajs.com/api/admin#orders\_postordersidfulfillmentsfulfillment\_idshipments)
API route.

## Example

```ts
sdk.admin.order.createShipment(
  "order_123",
  "ful_123",
  {
    items: [
      {
        id: "fulit_123",
        quantity: 1
      }
    ]
  }
)
.then(({ order }) => {
  console.log(order)
})
```
