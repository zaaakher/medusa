# registerOrderFulfillmentStep - Medusa Core Workflows Reference

This documentation provides a reference to the `registerOrderFulfillmentStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step registers a fulfillment for an order.

```ts title="src/workflows/my-workflow.ts"
import { createWorkflow } from "@medusajs/framework/workflows-sdk"
import { registerOrderFulfillmentStep } from "@medusajs/medusa/core-flows"

const myWorkflow = createWorkflow(
  "my-workflow",
  () => {
    const data = registerOrderFulfillmentStep({
      "order_id": "{value}",
      "items": [{
        "id": "id_EuhalJL2rhE6GvQMWa",
        "quantity": 3
      }]
    })
  }
)
```
