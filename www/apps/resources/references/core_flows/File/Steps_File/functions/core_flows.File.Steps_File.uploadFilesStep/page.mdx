# uploadFilesStep - Medusa Core Workflows Reference

This documentation provides a reference to the `uploadFilesStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step uploads one or more files.

```ts title="src/workflows/my-workflow.ts"
import { createWorkflow } from "@medusajs/framework/workflows-sdk"
import { uploadFilesStep } from "@medusajs/medusa/core-flows"

const myWorkflow = createWorkflow(
  "my-workflow",
  () => {
    const data = uploadFilesStep({
      "files": [{
        "filename": "{value}",
        "mimeType": "{value}",
        "content": "{value}",
        "access": "{value}"
      }]
    })
  }
)
```
